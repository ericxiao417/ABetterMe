# 排序算法总结

| 排序算法 | 最坏时间复杂度 | 最好时间复杂度 | 平均时间复杂度 | 空间复杂度 | 稳定性 |
| -------- | -------------- | -------------- | -------------- | ---------- | ------ |
| 选择排序 | n^2            | n^2            | n^2            | 1          | 不稳定 |
| 归并排序 | n^2            | n              | n^2            | 1          | 稳定   |
| 快排     | n^2            | nlogn          | nlogn          | n          | 不稳定 |
